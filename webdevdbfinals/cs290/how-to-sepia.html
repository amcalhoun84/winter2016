Fun with Sepia

Here, I created a map that is primarily done in a washed-out sepia style to show the versatility of the styling properties. Combine this with markers and other features we've discussed in this how-to, and you can create some pretty incredible maps with just a little bit of know how and experimentation.

The Code

<script type="text/javascript">

function initMap() {
var sepiaInLA = [
  {
    "elementType": "geometry",
    "stylers": [
      { "visibility": "on" },
      { "saturation": -75 },
      { "hue": "#ffaa00" }
    ]
  },{
    "featureType": "road.arterial",
    "stylers": [
      { "hue": "#ff8800" },
      { "color": "#ff8800" },
      { "saturation": -80 }
    ]
  },{
    "elementType": "labels.text.stroke",
    "stylers": [
      { "visibility": "simplified" },
      { "color": "#808080" },
      { "weight": 0.1 }
    ]
  },{
  },{
    "featureType": "water",
    "stylers": [
      { "color": "#2c1925" },
      { "saturation": -74 }
    ]
  },{
    "featureType": "road.highway",
    "elementType": "labels.icon",
    "stylers": [
      { "visibility": "on" },
      { "saturation": -74 }
    ]
  },{
    "featureType": "landscape.natural",
    "elementType": "labels",
    "stylers": [
      { "saturation": -63 }
    ]
  },{
    "featureType": "poi",
    "elementType": "labels",
    "stylers": [
      { "saturation": -33 }
    ]
  }
]

  var map = new google.maps.Map(document.getElementById('map'), {
    center: {lat: 34.0567, lng: -118.283},
    styles: sepiaInLA,
    zoom: 8
    
  });
}

</script>

You may notice a few little visual glitches or squares of color. This can be a side effect if you are not careful and I deliberately left them in to show what happens if you don't consider and handle every part of your map. If something like a city name or other label/symbol is not pertinent to your map, I recommend using visibility: off rather than risking it messing with the color or appearance of your map. Here, I've made ample use of the color, saturation, and hue properties. I selected the particular elements of the map I wanted to modify and went from there. The various labels and text were also modified to work with the map. See if you can take my code and make it even better!
